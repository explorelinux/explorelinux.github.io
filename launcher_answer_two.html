Follow the below mentioned steps to create a desktop icon on launcher.

  1. Go to the top-right corner of the desktop and click on the username(In this case it is Saketh).Then select "System settings".
  2. A window appears.In that select "Appearance".
  3. A window appears.In that select "Behaviour"(see the screenshot below).
  4. Now tick the check box-"Add show desktop icon to the launcher"
  5. Go to desktop and the violet coloured icon on launcher is the desktop icon.
