Advantages of Linux over Windows

  • Linux is free to install as it is an opensource software.To install windows we should purchase it.

  • The security aspect of Linux is much stronger than that of Windows.Why should we spend extra money on Anti-Virus?

  • In linux you have power to control every aspect of Operating system.

  • Because of its opensource nature regular users and programmers contribute applications all the time.

  • With linux you are not only getting the software for free but also you have an option to edit the source code to add additional features.

  • If an application terminates in Windows then it affects other applications(Sometimes it disables other applications).But in Linux all applications are independent.
