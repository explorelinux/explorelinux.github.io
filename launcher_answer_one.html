Follow the below mentioned steps to auto hide launcher in Linux.

  1. Go to the top-right corner of the desktop and click on the username(In this case it is Saketh).Then select "System settings".
  2. A window appears.In that select "Appearance".
  3. A window appears.In that select "Behaviour"(see the screenshot below).
  4. Now "ON" the button as shown in the screenshot.
  5. Select "Left side" option(Ignore if already done) and then go to desktop.
  6. Now move the cursor vertically in the place of launcher(left most part of desktop) to see the launcher.
