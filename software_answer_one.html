Follow the below mentioned steps to install an application using ubuntu software centre.

  1. Go to the top-left corner of the desktop and click on "Ubuntu Software Center"(See the screenshot).
  2. Now go to the top-right corner and type the name of an application which is to be installed.(See the screenshot)
  3. A new page is displayed.Now select an application(In this case it is VLC) and click "Install".
  4. Type the password and click "Authenticate".
  5. Now Wait for some time.Finally you can see remove option which means it was installed.
