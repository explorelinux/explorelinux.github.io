Follow the below mentioned steps to setup an alarm using Terminal.

  1. Open the terminal by typing "Ctr+Alt+t".
  2. Type "crontab -e" in terminal.
  3. A new page is displayed.(see the screenshot below).
  4. Now press "Down-Arrow key" to go to the bottom of the page.
  5. In the last comment(A line which starts with # is called as a comment),"m","h","dom","mon","dow","command" represent "Minute","Hour","Date Of Month","Month","Day Of Week" and "Command to be Executed" respectively.

  6. Now enter the particulars to open a music file(alarm) at this shedule.Range of Minutes is "00-59".Enter hours in 24 hours format i.e Range "00-23".Range of Date of month is "1-31".Range of month is "1-12".Range of day of week is "0(Sunday)-6(Saturday)".In command type "DISPLAY=:0.0 xdg-open song.mp3".

    "DISPLAY=:0.0" links terminal with GUI(Graphical User Interface) i.e Displaying default Media player(In this case it is VLC Media Player)."xdg-open" opens a file in appropriate format(Text in an editor(say gedit),music file in a media player(Say VLC Media Player))."/home/saketh/song.mp3" is the path of a music file.

    I have sheduled a task using crontab to open a music file(alarm) i.e song.mp3 on the 12th November,2015 at 5:15PM.

  7. Now type "Ctr+x" to exit.
  8. Then type "y" to save the changes.
  9. Finally press "Enter" to save the location(Please don't change the location).Crontab installs now.
  10. At the sheduled time(or with a delay around 20 Seconds) the music file(song.mp3 in this case) will be opened(See the screenshot below and notice the time on top right corner)
  11. If your system is off at sheduled time then type "anacron -n" before sheduled time so that the music file will be opened immediately when you turn on your system after sheduled time.
