Follow the below mentioned steps to create bookmarks.

  1. Click the files icon on launcher.
  2. Now open a directory which is to be bookmarked(In this case it is "Entertainment").
  3. After opening it,type "Ctr+d" to bookmark it.Now you can see the "Entertainment" directory on the left side under the heading "Bookmarks".(See the screenshot below)
  4. To remove a bookmark,"right click" on it(under the heading "Bookmarks") and select "Remove".
